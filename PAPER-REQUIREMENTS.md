# PAPER-REQUIREMENTS.md

This file provides guidelines to writing the report for this project (report/report.tex).

## Structure

* Cover Page
* Abstract
* Introduction > Motivation + Methodology
* Theory > Discussion of RL theory, different algorithms, and state of the art for the problem at hand
* Experiments > baselines, different algorithms (normal), different algorithms (flood-fill feature), two snake environment
* Discussion > analyze results, discuss challenges
* Conclusion > summary, future improvements
* References (all references included must be used in text)

## Guidelines

* Report must be populated with figures (long discussions are boring)
* No subsubsections (x.x.x)
* 20 - 30 pages
* Search papers using Websearch and paper-search MCP
* For every paper you cite, make sure it is reachable (verify with a subagent)
* Everytime you edit the report, run a critical reviewer to review
* The use of emdashes is disallowed. Replace appropriately
* All references must be used (in-text citation)
* No paragraph indentation (set noindent by default)
* No list item heading
* Prefer paragraphs writing over lists
